# Loan-Default-Risk-Analysis
This project is an end-to-end Loan Risk Analytics Dashboard developed using Power BI, SQL Server, and On-Premises Data Gateway.The dataset was stored in SQL Server, and Power BI was connected through an On-Premises Data Gateway to enable secure data access and refresh capabilities.

**Tools Used**

- SQL Server
- Power BI
- Power Query
- DAX
- On-Premises Data Gateway

**Key Features**

- Loan Default Analysis
- Borrower Demographics Overview
- Loan Status Monitoring
- Default Rate Tracking
- Risk Assessment Insights
- Interactive Filters and Slicers

**Project Workflow**

- Store and manage loan data in SQL Server.
- Connect SQL Server to Power BI using On-Premises Data Gateway.
- Clean and transform data using Power Query.
- Create DAX measures for KPIs and calculations.
- Build interactive dashboards and visualizations.

**Dashboard Insights**

- Total Loans
- Defaulted Loans
- Default Rate
- Loan Amount Analysis
- Borrower Demographics

Risk Trends and Patterns
Objective

The objective of this project is to analyze loan default patterns and provide actionable insights through an interactive Power BI dashboard powered by SQL Server and Data Gateway integration.
